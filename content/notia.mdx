# Notia

Notia is a note taking web app that uses markdown to format your notes and stores them in local browser storage.

<Image src="/projects/notia_home.webp" alt="Homepage image" />

## Features:

- Write notes in Markdown (CommonMark)
- Sort your notes by date and search
- Stored in browsers local storage

<Image src="/projects/notia_app.webp" alt="Homepage image" />

## What I used to build this

- React
- React Router
- Tailwind
- Vite
- Netlify
